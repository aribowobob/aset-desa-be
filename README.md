# Backend - Aset Desa
CRUD data aset pedesaan

## Installation
- Clone the project
- Go to `aset-desa-be` directory
- do `npm install`
- create `.env` file then copy-paste `.env.template` content. Adjust the value of environtment keys according to your local's setup.

## Run for development
- Go to `aset-desa-be` directory
- do `npm start`
